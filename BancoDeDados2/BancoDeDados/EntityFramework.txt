Anotações sobre EntityFramework - Code First
1 - Instalar o EntetyFramework - PM> Install-Package EntityFramework
2 - Crias as classe de modelo.
3 - Cria a classe de Contexto - Classe que vai representar o EntetyFramework e representar o Banco de dados.
	Define as Classes que vão virar tabelas no banco.
4 - Habilitar as migrações = Enable-Migrations
5 - Adicionar uma migração = Add-Migration NomeMigracao
6 - Atualizar o banco de dados = Update-Database -verbose


Vizualização do banco
1 - View / SQL Server Object Explorer

Desistalar EntityFramework

1 - desinstalar o EntetyFramework - PM> Uninstall-Package EntityFramework
